# خطة العمل: إنشاء شاشة "آفاق المستقبل والدعم" 🚀

تهدف هذه الشاشة إلى تعزيز التواصل مع الطلاب والمبدعين، وإطلاعهم على خارطة طريق التطبيق، وتوفير سبل لدعم استمرارية المشروع.

## 1. تصميم الواجهة (UI/UX)
*   **الهوية البصرية**: استخدام تصميم عصري يعتمد على البطاقات (Cards) مع أيقونات ملونة وجذابة لكل قسم.
*   **الأقسام المقترحة**:
    *   **خارطة الطريق (Roadmap)**: عرض المميزات القادمة (قاموس المفردات، مهارات لغوية، الاستماع مع التظليل) بأسلوب "Coming Soon" جذاب.
    *   **صندوق الاقتراحات**: نموذج بسيط لكتابة وإرسال الأفكار مباشرة.
    *   **تواصل معنا**: روابط التواصل الاجتماعي (فيسبوك، يوتيوب، واتساب) بلمسات عصرية.
    *   **دعم المشروع**: قسم خاص يوضح كيفية المساهمة مالياً (بنك فلسطين، PalPay، Jawwal Pay) لضمان استمرار التطوير.

## 2. المكونات التقنية
*   **HTML**: إضافة شاشة جديدة باسم `future-screen` في ملف `index.html`.
*   **CSS**:
    *   تنسيق البطاقات التفاعلية.
    *   إضافة تأثيرات Hover لروابط التواصل الاجتماعي.
    *   تصميم زر "نسخ رقم الحساب" لتسهيل عملية التحويل المالي.
*   **JS**:
    *   إضافة منطق التنقل إلى الشاشة الجديدة عبر القائمة السفلية (Bottom Nav).
    *   ربط نموذج الاقتراحات (سيتم إرسالها مبدئياً إلى البريد الإلكتروني أو كرسالة واتساب تلقائية حسب اختيارك).
    *   وظيفة "نسخ إلى الحافظة" لأرقام الحسابات البنكية.

## 3. تفاصيل المحتوى (بناءً على طلبك)
*   **المميزات القادمة**:
    *   قاموس مفردات الدروس والمهارات اللغوية.
    *   ميزة الاستماع الذكي مع تظليل النص المقروء (Highlighting).
*   **روابط التواصل**: (ستحتاج لتزويدي بالروابط الفعلية لاحقاً أو سأضع روابط تجريبية لتستبدلها).
*   **بيانات الدعم**:
    *   بنك فلسطين (رقم الحساب).
    *   PalPay & Jawwal Pay (رقم الجوال).

## 4. خطوات التنفيذ
1.  **المرحلة الأولى**: تعديل `index.html` لإضافة هيكل الشاشة الجديد.
2.  **المرحلة الثانية**: إضافة التنسيقات الجمالية في `style.css`.
3.  **المرحلة الثالثة**: تفعيل الانتقال للشاشة في `app.js` وإضافة وظائف النسخ والإرسال.
4.  **المرحلة الرابعة**: إضافة أيقونة جديدة في القائمة السفلية للوصول السهل لهذه الشاشة.
